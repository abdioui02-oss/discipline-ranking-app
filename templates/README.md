# Discipline Ranking App

A simple web app built with Python, Flask, HTML, and SQLite.

## Features
- Add users
- Prevent duplicate usernames
- Add discipline tasks
- Give points for completed tasks
- Show leaderboard
- Save task history
- Store data with SQLite

## Tools Used
- Python
- Flask
- HTML
- SQLite

## How to Run
1. Install Flask:
pip install flask

2. Run the app:
python3 app.py

3. Open in browser:
http://127.0.0.1:5000
